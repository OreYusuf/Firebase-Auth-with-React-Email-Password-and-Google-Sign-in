### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/OreYusuf/Firebase-Auth-with-React-Email-Password-and-Google-Sign-in
   ```

2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the project
   ```bash
   npm run start
   ```
